# 🎡 Spin the Wheel

An interactive browser-based spin wheel by **Toolsyte** for random selection, giveaways, classroom activities, games, and decision making.

🌐 **Live Demo:** https://toolsyte.com/tools/wheel/spin-the-wheel/

## Features

- Custom wheel entries
- Random winner selection
- Smooth spinning animation
- Mobile-friendly layout
- Browser-based tool
- Useful for giveaways, classrooms, teams, and games

## Use Cases

- Wheel of names
- Giveaway picker
- Classroom activity selector
- Random decision maker
- Team picker
- Party games

## Technology

- JavaScript
- HTML5
- CSS3
- Canvas / browser animation logic
