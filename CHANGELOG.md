# Changelog

All notable changes to the **Spin the Wheel** project will be documented in this file.

The format is inspired by [Keep a Changelog](https://keepachangelog.com/), and this project follows semantic versioning where appropriate.

---

## [1.0.0] - 2026-07-03

### Added

- Initial public project documentation.
- Interactive browser-based spin wheel.
- Unlimited custom wheel entries.
- Smooth spinning animation.
- Random winner selection.
- Mobile-friendly responsive interface.
- Built-in presets for quick wheel creation.
- Entry shuffle feature.
- Alphabetical sorting.
- Duplicate removal.
- Adjustable spin duration.
- Winner celebration animation.
- Local browser storage for saved entries.
- Clean and accessible user interface.

### Documentation

- Created GitHub repository.
- Added project overview.
- Added feature documentation.
- Added project screenshot.
- Added roadmap section.
- Added repository topics.
